# optimustech
Esse projeto é o resultado do desafio da "7days of code", com parceria com a Alura, sobre html e css. É um site que simula um RH de uma empresa, mostrando cargos, quem é a empresa e entre outros. Eu utilizei a tecnologia flexbox para organizar o layout da página, e usei também as unidades de medidas relativa como "rem" e "em".
